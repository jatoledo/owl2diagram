# Owl2diagram



## How to use

1. Clone the Owl2diagram repository

   `git clone https://github.com/jatoledo/owl2diagram.git`

2. Run the docker image with docker-compose

   `docker-compose up -d`

3. Starting server at http://127.0.0.1:8000/
